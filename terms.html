VolvoUWBUnlock - Terms and Conditions

Last updated: February 12, 2026

1. Overview

VolvoUWBUnlock is a personal research application developed by the vehicle owner as part of a personal AI agent (LLM-based) research project. The application uses a personal UWB (Ultra-Wideband) accessory and the owner's iPhone to detect proximity to the vehicle, triggering a workflow that calls the Volvo Cars Connected Vehicle API to unlock or lock the vehicle.

The UWB proximity detection is independent from the vehicle's own systems and does not connect to or interact with the car's hardware.

2. Personal Use

This application is intended solely for personal use by the vehicle owner. It is not distributed to or intended for use by third parties.

3. Data Handling

All vehicle data obtained through the Volvo Cars Connected Vehicle API (including door/lock status and location) is stored exclusively on the owner's device. No data is collected, transmitted to external servers, or shared with any third party.

4. Third-Party Services
